数据线

最近研究了一下USB-C口的数据线，发现门道不少……

USB数据传输标准一直在更新，最新版本已经到了USB 4.0。目前用得比较多的还是USB 2.0和3.x这两大系列。

USB 2.0系列

最早还有USB 1.0和1.1，这个太古老了，就不讨论它们了。

在这里提一下「传输损耗」这个词，我们在一些datasheet里看到的传输速度值，都是「理论值」，实际上是有损耗和衰减的。比如USB 2.0的数据传输速率是480Mbps，也就是480/8=60MB/s。但通常来讲，它的实际传输速度能到40MB/s就很不错了。当然这个跟USB设备能否支持这个传输速度也有关系。比如一个U盘，它的芯片达不到40MB/s这个速度，只能到15MB/s，虽然也能称之为USB 2.0，但还是需要注意的。

USB 2.0的接口是长方形的，里面的触点是4个，从侧面看是白色的。这种数据线非常常见。

USB 3.0接口

USB 3.0开始，接口规范就多了起来。

USB-IF协会首先推出了USB 3.0，后来改名成了USB 3.1 Gen1；

然后又出现了USB 3.1 Gen2和USB 3.2。

现在市面上USB 3.2的线也很少见，更多的是3.1的那两种。

更让人晕头转向的是，现在USB-C接口的线越来越多，而且是能够兼容USB 2.0、3.1 Gen1和Gen 2这些协议的，

也就是说：你买到的USB-C线，有可能是上述三种标准之一，看着很漂亮的线，实际上是2.0的传输速度，恶心不？

供电能力

USB接口发展至今，除了传输速度有改进之外，对于电气方面也有了不小的提升。

USB 2.0接口能支持500mA的电流，5V电压；

USB 3.1 Gen1能支持900mA的电流，5V电压；

USB 3.1 Gen2能支持5A电流，电压可以支持5V/12V/20V，最大供电达到了100W。

现在手机、电脑设备都上了USB-C接口，随着各种快充的普及，对于USB-C线缆的要求自然就高了。

首先要明白一点，通常来讲，如果设备要支持快充，数据线插头可能是Type-A，也可能是Type-C的。当然现实中还是TypeC居多的。这样做的好处是，不需要对线缆有什么变动，只要符合标准，就能支持快充。

早期的一些手机厂商，为了推广自己的快充协议，在数据线上做了改造，必须用配套的数据线才能激活专有的快充协议。如今看来似乎没什么必要，因为真的不差那几分钟。谷歌也要求近两年生产的安卓手机必须兼容PD快充协议，这样就挺好，没有一个能统一的快充协议，吃亏的还是消费者自己。

快充协议

市面上的快充协议林林总总，现阶段只要关注PD快充协议即可。

PD协议目前最新版本是3.0，已经涵盖了
