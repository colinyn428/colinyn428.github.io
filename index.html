我的Mac软件分享

用Mac系统也半年多了，结合工作需要（运维相关），分享一些比较常用且好用的软件。

  1. Microsoft Remote Desktop Beta

    微软的远程桌面连接程序，是免费的。顾名思义，ONLY for Windows。界面无中文，不过用起来无难度。

    更新比较频繁，经常修复bug，符合微软的风格。

    下载地址

  2. Typora

    非常棒的MarkDown写作软件，支持即时渲染技术，预览起来更直观。本博客由于使用了Hexo框架，直接支持MarkDown格式，所以用Typora编辑就更加方便。

    官网

  3. Microsoft Edge 浏览器 和 Google Chrome 浏览器

    这两个浏览器都是Chromium内核，因为某种原因，Edge的同步功能更好用。内存占用方面Edge更占优势一些，流畅度方面Chrome更好一些。总之不用Safari就是了。

    Edge官网 ｜ Chrome官网

  4. 腾讯会议

    在家办公，开会必备软件。目前还是免费状态，语音通话的清晰度不错，视频会议很少用，屏幕演示功能很实用。

    官网

  5. UltraEdit

    知名的纯文本编辑工具，Windows平台也有。主要用来去除各种文本的格式，退出前如果没有保存文档，再次打开是看不到的，这点不如Notepad++，不过Mac平台上没有后者，遗憾。

    下载地址欠奉，请自行搜索。

  6. Xnip

    简单好用的截图+标记工具，除了常用的窗口/全屏截图之外，还支持滚动长截图，虽然有水印，但不会打在正文上，非常良心。

    官网

  7. Mos

    Mac系统如果外接了鼠标，那么滚轮的方向跟触控板正好是反着的，并且滚动起来也不平顺。有了Mos这个小软件，就能达到接近触控板的平滑效果，滚动方向也能回归到习惯的方式。另外还可以设置例外程序。

    官网

  8. Hidden Bar

    菜单栏管理器。Mac系统顶部的菜单栏是左边的菜单和右边的托盘图标合二为一的，托盘图标很多时，看着就很乱。这个软件就有点类似Windows系统的「系统托盘」，只要按住⌘键并拖动图标到竖线左侧区域，这个程序的图标就会被隐藏。

    下载地址

  9. Magnet

    窗口管理工具。当外接显示器时，如果分辨率较高（比如达到了4K），那么系统自带的「将窗口拼贴到屏幕左侧/右侧」这个功能就显得很鸡肋，因为通常都要打开很多个窗口嘛，因此Magnet这类窗口管理工具就非常实用。

    可以看到，Magnet不仅提供了常见的两分、三分和四分布局，还贴心地为高分屏提供了六分布局。如果觉得点击窗口再点菜单来调整太麻烦，那就建议记住快捷键，效率更高。

    下载地址（收费，18元）

  10. 小时钟

    一个集成了时区和倒计时的小程序。程序打开后会运行在程序坞里，可以添加不同时区的城市。像我们公司领导目前人在美国，添加一下那边的时区，沟通时也能做到心里有数。另外吐槽一下Mac的时间设置，只能设置一个时区时间，太抠了。

    倒计时的时候可以设置提示音，也挺方便的。

    下载地址

  11. iPaste

    剪贴板管理工具。免费版只能记录最近的3条剪贴板历史，不仅仅能管理文本剪贴板，还能管理图片剪贴板。对我来说免费版就够用了。

    下载地址（专业版50元）

还有一些软件也很好用，比如FreeSync，Downie 4等等，打算以后再建专题慢慢说。

开学前三周

经历了史无前例的网课一学期和独自带娃的多半个暑假，2020年终于正式开学啦！犹记得9月7日当天的朋友圈，各种晒娃去学校的，「神兽」们终于被送回他们该去的地方了，家长们都激（kai）动（xin）不已。

开学后，孩子的作息时间跟之前差不多，不过因为没有课后兴趣班，所以放学早了些，下午3点就要接回来。多亏有老人帮忙照顾，要不然每天的吃饭问题也是挺头疼的。学习方面是先复习一遍上学期的内容，然后是上新课。感觉娃对于新学期的接受能力也挺强的，每天挺开心的，各方面也可以说是无压力。

一直到第二周周末都是非常顺利，直到周日傍晚，娃开始头疼，吐了一回，睡醒一觉以后发低烧，不能去上学了。只好请假让他在家休息。经分析应该是有点着凉导致的，老婆去药店买药，还挺对症，目前为止基本上是恢复了。小朋友的体质虽然比之前强了，但还是扛不住头疼脑热。加上现在疫情防控要求，各方面都更加谨慎从严。本着「好利落再去学校」的原则，把假请到了本周五，周日再去上学吧。

一个星期虽然废了，在家也还是要坚持学习。英语课、数学课的作业该做也都做了，其他课外班的作业也在努力完成。总体来说娃还是挺听话的，现在已经会帮忙蒸馒头蒸鸡蛋了，继续努力，离做饭不远啦，加油！

纯电动汽车之我见（三）

使用及售后

先把售后放前面吧，内容不多但是挺重要的。

「质保」

首先是「保养」。纯电动汽车的保养是真的很简单，没有汽油车那么复杂的保养项目，只是检查检查插线，清理灰尘就OK了，费用也很低廉，像北汽对于EV200这款车的政策是前四次保养免费。

其次是「质保」。要分清楚是「整车质保」还是「电池质保」。通常来讲电池质保的时间或里程是较长的，但是质保时有没有费用，或者有没有可替代的备件，都说得很含糊。电动汽车的电池价格很贵，基本上是整车价值的一半或更高，它的更新速度也是较快的，有可能到时候连电池都没有了。那样的话，这车只能报废了。

「二手与回收」

纯电动车因为是个新生事物，二手市场目前对它的接受程度普遍不高，能接受的也是特斯拉这种大牌子。如果厂家回收的话，一辆10w左右的车，2年下来顶多能卖到原价的一半。4、5年的车根本卖不到2w元。所以车主买车时很开心，换车时很郁闷。

「使用感受」

从我和我老丈人这些年的驾车感受看，从E150EV到EV200再到EX360，故障率总体来看是越来越低了。

E150EV的故障率是真的高，而且不仅仅是电动方面的故障，还有汽车其他配件的问题。比如副驾驶玻璃会在冬天掉下去，原因是天太冷，把车门里的卡子冻坏了……而且不是个例！我不会忘记那年冬天穿着大棉袄开车去修理厂的情景！

EV200遇到过一次蓄电池接触不良导致没法着车，所以车里一定要常备修车工具。

EX360总体还是不错的，但是感觉电池没有EV200好，我用快充次数比较多，从续航读数上也能感觉到电池容量有衰减。所以电池越大，一旦有衰减就感觉很明显。

「影响续航的因素」

首先厂商宣传的都是NEDC续航，这个数值很虚，具体介绍请见这里。而实际工况比这个要复杂很多，因此基本上要用这个数值乘以0.8才比较靠谱。

影响续航的因素大概有这么几点：

  • 车速——通常60Km/h是最省电的，速度越高越费电。低速蠕动也相对耗电，但比跑高速要好。车辆刚启动时的电耗也是较高的，速度起来以后电耗就会慢慢降下来。

  • 车重——这个很好理解，不解释了。

  • 空调——开暖风最费电，开冷风反而对整体续航影响不大。跟空调的制冷/制热功率有关。

  • 环境温度——如果气温在5℃以下，锂电池的活性会降低，总电量会减少，因此会影响续航总数。

  • 急加速急刹车——电机需要瞬时提高功率，如果是这个驾驶习惯，对续航也会有影响。

  • 是否堵车——如果堵车时没开空调等费电的设施，那么基本上不费电。车速为0停下后电机就不介入了。

  • 是否开启车窗——行驶途中全关车窗会降低风阻，提高续航。但是跟车身的设计有关。

这些因素中，对续航影响较大的是车速、空调和环境温度。如果你是冬天又开空调又上高速，那最大续航基本上就是NEDC的一半。一定要有心理准备！

当然，比较佛系的驾驶习惯对续航也有一定帮助，这样相对也更安全，不推荐频繁急加速急刹车。

还有一个个人经验，就是车里显示的剩余续航不见得准确，可以根据「百公里耗电度数」来进行换算，得出一个大概的续航里程。

比如一辆车是45度电，表显百公里耗电数是13kWh，那么这辆车理论上能跑45÷13×100=346km。

纯电动汽车之我见（二）

充电

开纯电动车，就一定要去充电。

E150EV那会儿是纯电动车刚起步的时代，做的厂商不少，百家争鸣，但是并没有一个统一的充电标准。那会儿充电桩都很少，不同品牌的车用同一个充电桩充电，都可能会出现不兼容的现象。我那会儿还是用慢充比较多，也就是从家里的16A空调插座取电去充。那个时候真的遇到了很多问题，比如空调插座没接地线导致充不了；电线或者空开质量不过关导致充一半断了；还赶上过充电宝坏了……总之克服种种困难，最终还是把电给充上了。

这些年随着国家的大力扶持，资本的介入，以及充电协议国标的颁布与实施，充电桩如雨后春笋一般冒了出来，不兼容的问题越来越少，与智能手机的整合也越来越密切，使用体验有了质的提高。充电协议国标是2016年1月1日颁布的，所以2016年以后生产的纯电动车都会兼容这个标准。我是快充、慢充都用过，总体区别大概是这样：

  1. 直流快充：9个触头，额定电压电流是750V 125A/250A，但实际上达不到。这个主要跟充电桩所在的电网资源以及电池参数有关。我目前用EX360充电，表显最高大概是340V/120A左右，功率在40kW左右。如果很多车同时在充电，这个数值就会有波动了，可以理解为「抢资源」。我的车从30%到充满电大概1个小时左右。快充桩附近一定有小的变电站设施，充电线缆是粗又硬（你懂的），力气不够的确实要费点儿劲。
  2. 交流慢充：7个触头，比直流快充个头要小。我见到的多数是380V/32A，也有380V/16A的。如果有车位，安装的充电桩也是这种，电压可能不同。我的EX360从30%到充满电要3个小时左右。慢充桩的线缆就细多了，形态也比较多元化。
  3. 交流充电宝：这个一般是随车附带的，也是7个触头，是交流慢充的一种，由插头，充电枪和变压器组成。电路规格是220V/16A，必须插在16A的空调插座上使用。我的EX360是48度电，充满大概要8-10个小时。

曾经看到有人说纯电动车最理想的充电场景就是在自有的固定车位用交流慢充来充电。现在想想真是很苛刻，物业不让装充电桩的还是多数；走民用电表还是企业电表也不是消费者说了算的；安装充电器材的隐性消费也不少……

在充电过程中，还有一些问题是车主一定要面对的，比如：

充电费用问题。如果充电桩走的是居民用电，一度电0.5元，那一切都非常舒服：充满48度电也只需要24元钱，能跑300公里，平均百公里只要8元钱，比坐地铁还划算。而如果是工业用电，那就更高一点儿；如果去充电桩充电，这个价格就要比居民用电贵至少一倍。以我们用的最多的国家电网充电桩为例：

国网充电桩分为「峰、谷、平」三个价位，分别代表「高峰时段（8:00-11:00，18:00-23:00）」、「低谷时段（23:00-7:00）」、「平时段（7:00-8:00，11:00-18:00）」，各时段的充电费用是不一样的，从1.8元至1.2元每度电不等（含服务费）。

而其他品牌的充电桩，充电费用也从1.2元-2元每度电不等（含服务费）。所以我一辆48度电的车，充满电可能最多要花96元钱！这还不包含停车费。可见如果经常去充电桩的话，用车成本并不低。

还有一个不能忽视的问题就是冬天充电。由于锂电池的特性，北方的冬天对它来说就是灾难（参考iPhone冬天自动关机）。不仅续航里程要打6折，充电时间也变得更长。因为锂电池低于5摄氏度，锂离子的活性就会大幅降低，影响充电效率。所以你会看到冬天充电时，充电桩的功率都比较低，因为电池温度上不来。所以解决方法就是尽量「热车充电」，车子跑了一段路以后直接去充电，不要等车温度凉了再充。

还有其他「小」问题：比如汽油车占了充电桩的车位啦；车子充完电后不开走啦；好不容易找到充电桩却用不了啦（坏了或者这是公家的不让用）……这些问题多半是管理方面的，相信如果想解决的话，一定能有妥善的方法去改善。

纯电动汽车之我见（一）

本人目前是北汽新能源EX360的车主，曾经开过北汽E150EV，EV200，算是比较早接触纯电动汽车的人。写一写这个系列的目的，主要是想谈谈我对纯电动汽车的一些浅见。

初见纯电动汽车

我2011年拿的驾照，期间一直没碰过车。第一辆车大概是2015年前后吧，是北汽E150EV。因为北京政策的原因，当时汽油车摇号中签率就已经非常低了。随着孩子的出生，有辆车的需求也逐渐迫切起来。那会儿也考虑过租别人的号牌，甚至还打听过外地牌照，当然都一一否决了，握在自己手里的才是最靠谱的，最终还是不得已选择了纯电动车。现在北京的政策也能看出来，混动/插电依然不算新能源，更加限制外地车，新能源摇号更是一号难求，都排到2027年了。回过头看看，当时的决策非常重要，否则车都开不上。在这里感谢老丈人！

与汽油车的驾驶感受对比

在买车之前我找过几次陪练，不过目前现在为止开汽油车的次数也是屈指可数。主要对比如下：

  1. 续航短。那会儿的纯电动车真的只有150KM的续航，天冷的时候充完电只有133KM。如果赶上冬天+开暖风，那续航直接按50%算吧。
  2. 0-50提速快。即使是纯电动车比较低端的E150EV，0-50KM/h的提速也很快。加速踏板响应非常直接，一点儿也不「肉」。那时候我还是新手，不敢猛加速。但是因为电机功率的限制，功率越大耗电越多，所以厂商限制了50KM/h以上的加速性能。开这个车在五环上超车还是要掂量着点儿。
  3. 刹车不同。我平时坐出租车，司机会在离前车比较近的距离时才踩刹车。不过纯电动车就不同，它有一个「动能回收」的功能，刹车会被分成两段，前半段踩的时候是动能回收阶段，会把车辆从快速到慢速这个阶段的动能保存起来（你可以理解成它不再分配动能了，而是把这些能量收回去了）。所以你踩的时候会觉得车速稍微减慢一些，但是刹车的感觉并不强烈；后半段才是刹车系统介入，这时候能感觉到刹车钳和刹车片的摩擦，车辆快速减速。实际上只要你在汽车行驶途中松开了加速踏板，车辆就在进行动能回收。据说跑山路长下坡的时候动能回收效果会很好，不过一直没有机会尝试。
  4. 车重增加，底盘低。因为电池都被安装在车辆底盘中间的位置，所以车重会增加不少。我记得E150EV这种小车车重大概都有1.3吨，车重的增加会让高速驾驶时车身更稳，不会觉得很「漂」。
  5. 价格普遍比燃油版贵。传统车企做纯电动车一贯的套路就是「换汤不换药」，就是外壳不怎么动，把里面燃油车那一套东西换成电动版，内饰稍微小改一下就拿出来卖了。因为纯电动车最贵的部分是电池，而且由于国家有巨额补贴，所以赚得最多的一定是车企。一辆E150燃油车卖6-7w，纯电动版的价格至少要增加1/3，就得卖9-10w左右。最让人受不了的是内饰依然简陋得不行，除了仪表盘、挡把这些有所改动，其他部件原封不动照搬燃油版。
  6. 补能不方便。这在当时不仅仅是充电慢，还有充电桩稀少的问题。关于充电我会在后续章节里细说。

独自带娃两个月

独自带娃两个月

娃的暑假生活已经过去了一半，之前一直是玩玩玩的状态，从7月底开始就迎来了各种课外班，为了保证学习效果，还是决定把他从姥姥家接回来，由我进行全程监督。我从疫情爆发到现在一直在家办公，工作负荷量尚可。现在不知不觉已经带了他两个星期了，深刻体会到了全职老爸的感觉，每天两次直播课，期间还包括十分烧脑的奥数课；中午还要琢磨吃什么，对于很少下厨的我来说，真的感觉身体要被掏空了，身心俱疲……😂

不过事情一定是有两面性的：首先学习效果有保证；然后亲子关系也会更好；最头疼的吃饭问题也解决了：媳妇回来做点儿，第二天我们就热着吃了，我有时候也做饭吃，要么中午去他奶奶家吃，要么就点点儿外卖。总之各种方法轮番上阵，半个月时间还挺快就过去了。

对于娃的进步，主要提奥数课和足球课：

奥数课

小学二升三的奥数题吧，计算并不难，难的是解题思路。对一般人来讲，首先要理解题意，这个在任何科目里都是非常重要的，因此语文不能差。然后解决这个问题，需要分成几步，每步要怎么做，也要非常清楚，才能把题做对。用老师的话说就是「思维链条要长」。校内那种「一步两步出结果」的解题方式，在奥数这里基本都是三步起。娃算是头一次接触奥数的萌新，很多地方学得确实没那么透彻，跟班里的其他学霸比起来有一定差距。曾经看过一篇文章，大概意思就是「适合学奥数的学生也就是5%」这样，但这并不等于说就不学奥数了，我们需要的还是解题思路，对于一个问题的不同的发散思路。

通过上这10节奥数课，我能感觉到他的进步有：

  1. 计算速度有提高。二年级下半学期因为都是在家上网课，基础一定没有在学校时打得牢，所以提高或者说保持计算能力非常重要。奥数的每一步计算对他来说都是一个积累和复习的过程，所以无论是口算还是笔算，都有进步。
  2. 笔记技能有提高。对于老师PPT上的板书，虽然没法做到100%都抄下来，但还是很努力地在写，我觉得这就是进步。一方面写字速度快了，另一方面也能分辨哪些是重点了。
  3. 举一反三能力有提高。奥数课大部分题都是「例题+练习配套出现」的模式，例题弄明白了，练习基本上也能做出来。他偶尔还会自己给自己出题，这也是提高的一个表现。

当然缺点也一定是有的，上述几点的反例他也都存在，我也希望能引导他能做得更好，这也需要时间去调整。

足球课

受疫情影响了半年多的足球课终于重新开课了，半年多没见面的小队友们又可以在一起愉快地踢球了！

我们报的这个足球课属于非常佛系那种，多数家长对孩子的期许就是「以锻炼身体为主」，而不是以打比赛为主那种。所以这也算是个双刃剑吧，训练过程很轻松，上课氛围很轻松，待改进的地方就是队员基础参差不齐，训练效果一般。有一些孩子练了1年了，确实有进步，但是似乎有限？

目前上了三四次课，总体来看娃的状态是稳步回升的。第一次课回来累得不行，现在已经能适应了。通过这几次课的观察，他敢抢球敢射门了，与队友们的关系也更融洽，也有了一定的队伍荣誉感。

每次训练的最后环节都是分组对抗比赛。小球员们都非常期待比赛，当然也还是比较看重胜负，如果有一方被压制或者输球，能明显感觉到士气低迷。不过每到半场结束，都会重新分配一次场上队员，也能起到一定作用。娃在队里被分配比较多的位置是前锋，现在踢球也稍微聪明些了，从之前满场瞎跑到现在知道找位置了。随着进球数量的增多，也能感觉到他的自信增多了，一开始不敢抢球，现在也敢伸脚了；一开始不敢张嘴要球，现在也会喊话了；一开始不敢射门，现在也知道起脚了。当然结果还是有待提高的，但有了这些意识，相信今后也会有质变的。

每次比赛结束做拉伸的时候，教练都会强调「胜负得分不重要，重要的是过程」，我很认同这句话，如果不注重过程，一味地追求比分，那才是真的害了孩子。

前两天得知9月7日开学，朋友圈里家长们都开心得不得了。我觉得在家学习和在学校学习是两个不同的氛围。对于在家独立带娃，虽然累，却也是个难得的体验，目标都是为了让孩子成长得更好，我很享受这个过程。
